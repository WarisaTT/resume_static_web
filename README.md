# Warisa Thiamthong | Professional Portfolio & Resume

A high-performance, responsive, and visually stunning personal portfolio website built with modern web technologies. This project serves as a comprehensive digital resume, showcasing professional experience, technical skills, and key projects.

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen.svg)](https://warisatt.github.io/resume_static_web/)

## ✨ Key Features

- **Responsive Design**: Optimized for all devices, from large desktop monitors to small smartphones.
- **Multi-language Support**: Seamlessly toggle between English (`index.html`) and Thai (`th.html`).
- **Dynamic Theme System**: 
    - **Dark Mode**: Sleek, high-contrast design with glassmorphism elements.
    - **Light Mode**: Clean, warm, and highly readable professional aesthetic.
    - **Theme Persistence**: Uses `localStorage` to remember the user's theme preference across sessions.
- **Interactive UI**:
    - Animated "Hero" section with a floating profile frame.
    - Profile Image Carousel that cycles through professional and casual photos.
    - Modern tech-tag system and glass-card layouts.
    - Smooth scroll navigation and reveal animations on scroll.

## 🛠️ Built With

- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3 (Vanilla)**: Custom styling without heavy frameworks, utilizing CSS variables and modern layout techniques (Flexbox & Grid).
- **JavaScript (Vanilla)**: For theme switching, image carousels, and scroll interactions.
- **Font Awesome**: iconography.
- **Google Fonts**: 'Outfit' for English text and 'Noto Sans Thai' for Thai content.

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/WarisaTT/resume_static_web.git
   ```
2. Navigate to the project folder:
   ```bash
   cd resume_static_web
   ```
3. Open `index.html` in your favorite browser.

